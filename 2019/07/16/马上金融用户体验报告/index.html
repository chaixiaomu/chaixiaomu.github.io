马上金融用户体验报告

发表于 2019-07-16

体验环境

手机：iPhone7

系统版本：iOS 12.3.1

APP版本：马上金融 V4.10.25

体验时间：2019年7月15日

分析目的

  • 探寻消费金融行业的商业模式和市场格局
  • 分析马上金融 App 产品的功能及用户体验
  • 针对此次体验，提出一点改进意见
  • 总结消费金融行业的商业模式并得出思考

消费金融市场现状

在进行产品分析之前，笔者认为有必要对目前消费金融行业的情况进行简要分析（经济学上常把投资、消费、出口比喻为拉动GDP增长的“三驾马车”，这是对经济增长原理最生动形象的表述）。可以总结为五点：

  • 强监管
  • 蓝海万亿级市场
  • 稳格局（1银行系+4非银系）
  • 金融持牌机构将迎来重大机遇
  • 借力科技，覆盖多场景

强监管

行业监管不断规范，各类业务逐步纳入金融监管体系。经历了近几年互联网消费金融集中爆发的风险外溢事件之后，类似校园贷等分期购物平台遭遇到了政府监管的强监管，针对大学生的校园贷被国家明确取缔。对于行业发展来说，此时是典型的去泡沫去杠杆时期，虽短时间内行业发展会遭受挫折，遇到“寒冰期”，但对于未来合规化、成熟化的行业发展和市场运作是必要和有利的。

蓝海万亿级市场

消费信贷前景可期

在经济下行的压力下，以往三驾马车主要依靠投资，而现在则转向消费驱动。从核算角度看，2018年前三季度最终消费支出对经济增长的贡献率是78%，比上年同期提高了14个百分点。

消费有增长空间，消费金融企业就有发展。《前瞻产业研究院》显示，当前我国居民消费在GDP中所占的比重在35%以下，而美国该数值在60%以上，日本也维持在50%，数据上的差异表明我国当前经济结构发展尚未成熟，经济增长方式从投资拉动、外贸驱动逐步转向消费带动还有很大的转化潜力。此外，目前国家共发了放了28张消费金融牌照，消费金融企业中消费金融公司就占了24张。无论是从供给还是需求任何一方来看，消费金融企业前景可期。

信贷需求难以被满足

央行个人征信记录收录了8.6亿人，但有信贷记录的仅占一亿多人，传统银行等金融机构给贷款申请者放款审批时，通常会参考申请者在央行的个人征信记录，从而判断是否放款。而央行无征信人群基数较大，该群体的消费信贷需求难以被满足。

无央行征信人群覆盖群体基数较大，随着消费需求的增加、消费升级与消费结构的变化，传统食品、衣着等支出逐渐下降，人们更愿意将钱花在享受和投资上。休闲娱乐、医疗护理、3C等数码产品的占比逐渐增加，由此催生了这些领域的消费金融的快速发展，但其坏账率明显高于行业平均水平。

清华大学中国与世界经济研究中心主任、中国经济思想与实践研究院院长，李稻葵表示，未来中国消费金融行业发展前景很好，2019年的关键字是“变”，消费金融相关政策已经出现了本质性的、方向性的调整。随着中国消费对经济的贡献度增加、消费贷款在贷款总规模盘子中比重增加、消费贷款中房贷、车贷等中长期贷款比重降低，消费金融公司可以凭借其自身优势大力拓展与商业银行错位发展的无抵押、无担保消费贷款，使无抵押、无担保消费贷款在整体消费贷款中比例大幅上升，消费金融公司将迎来万亿蓝海市场。

稳格局（1银行系+4非银系）

产业结构

互联网消费金融产业结构主要包括资金提供方、服务商与消费群体，除此之外银监会负责监管，征信与大数据提供信用报告，催帐机构催收欠款。

市场格局

目前消费金融领域的格局为：1银行系+4非银行系，主要分为垂直分期购平台、电商类消费金融、消费金融公司、银行机构互联网化以及P2P平台。截至目前，互联网消费金融在不同领域迅速发展，并形成了几种代表性的模式，一类是主打分期购物互联网消费金融平台，如趣分期、乐分期等；一类是电商平台的消费金融，以蚂蚁金服、京东金融、苏宁金融为代表；一类是P2P平台，以人人贷、拍拍贷等为代表；一类是消费金融公司，以中银消费金融、马上金融等为代表；一类是以银行为代表的传统金融公司互联网化，比如招行消费金融等。

  • 垂直分期购平台，根据应用场景的不同，分为不同类分期购平台，深耕细分领域，融资成本高。
  • 电商类消费金融，依托自有消费场景，拥有稳定、高消费能力的用户。
  • 消费金融公司，银行与产业的强强联合，更具普惠金融特征。
  • 中银消费金融，主要业务包括信用贷款、商户专享贷和互联网贷款。
  • P2P平台，以现金贷为主，野蛮式成长之后将迎来行业与法律法规重点监管。

金融持牌机构将迎来重大机遇

强监管下，消费金融持牌机构具有先发优势。当前互联网消费金融行业正处在监管的风口浪尖，按照监管做金融必须持牌的要求，银行和消费金融公司等持牌机构将会成为互联网消费金融的主角，由于银行的受众群体和风险风控能力承受度较低，消费金融公司无疑成为最大赢家。

2017年下半年，消费金融公司起开了一波增资潮，“增资潮”就是在增加企业总资本，提高资本充足率，由于通过提高实缴资本，借贷额度、同业拆入资金的额度也会提高，从而达到加大覆盖率、扩大市场份额的作用。与此同时，纷纷增资也代表了消费金融公司普遍看好后续市场，以此方式加大投入做强做大，从而抢占市场份额。

借力科技，覆盖多场景

从调查数据来看，消费信贷的用途较为宽泛，满足了消费者在各个方面的消费需求。相比比重方面，买家电占比为26%，教育培训、旅游、家庭装修和非车交通工具比重分别为16.4%、10%、13.6%和7.8%。

金融消费多场景的发展，离不开科技的助力。消费金融企业目前正在运用互联网技术，创新金融服务手段、提升金融行业资源配置效率；并利用互联网+等技术手段，优化金融服务渠道、创新手段，通过手机、APP、网上直销、电子商务平台等渠道，构建线下网点和线上补充的业务网络体系。

正是在科技的支撑下，消费金融得以覆盖更多场景，目前消费金融公司的布局主要集中于商业银行“无暇”顾及的细分领域，范围涵盖大众消费、婚庆、家政、养老、健康消费、信息消费、旅游休闲消费、教育文化体育消费等消费热点领域。

业内人士认为，在新监管和去杠杆形势下，通过不断开发消费新场景并嵌入金融服务将是消费金融公司主要发展方向。随着互金平台陆续退出，持牌机构头部公司则需要不断规范自身行为强化龙头效应。未来，科技手段提高风控能力有望成为消费金融公司的护城河，利用降低逾期风险等方式提高平台收入利润，并依靠细分领域消费场景，获得多元化获客渠道。

用户分析

除消费不断增长外，消费者的消费行为正在不断改变。80、90后已逐渐成为主要消费人群，与传统消费群体不同，新兴消费群体的消费观念更加超前，价格已不再是决定消费的唯一因素，借贷观念更易被接受；此外，受消费升级影响，三、四线等小镇城市消费人群的消费水平也在不断提高。

清华大学中国经济思想与实践研究院发布《2018中国消费信贷市场研究》（下称“报告”）显示，居民消费信贷的客户18岁至29岁的在校及刚毕业的群体占比36%，是几大群体中占比最多的。30—39岁群体占比为30%，40—49岁群体占比为33%，50岁以上中老年群体则很少有使用消费信贷的。

在收入分布上，月收入在2000—4999的群体占样本总数的70%，其次为5000—9999元群体，2000元以下低收入群体和10000元以上高收入群体里消费金融行为较少。

在教育程度方面，本专科生所占比例最大，专科生占比57.5%，本科生占比24.5%。一方面是因为本专科生毕业后与研究生以上学历群体相比工资相对较低，另一方面也可能是他们接受了较高水平教育后对消费信贷方面的了解比高中以下群体更多。

调查结果表明职业和收入较为稳定的企业员工和公务员事业单位人员是消费信贷的客户主体，占比达到84%。

著名经济学家、北京大学经济学院特聘教授陈志武曾发文表示：“消费金融的主体是年轻人才符合经济学规律。对于老年人甚至很多中年人来说，消费金融的意义不是太大，他们不需要这些金融工具的支持，而年轻人有很强烈的消费需要，用一些金融工具很正常。老年人是不需要消费金融支持的，因为他们一辈子的收入潜力都已经实现得差不多了。年轻人未来的收入潜力巨大，所以，消费金融在相当程度上是针对年轻人的。”

马上金融产品简介

官方产品简介

马上金融APP是由马上消费金融股份有限公司（以下简称“马上消费金融”）研发搭建的信贷平台，平台上已有产品包括马上贷，马上钱包，商品贷等。马上消费金融致力于为中国国内居民提供更为便捷的个人消费金融服务。

  1. “马上贷”是一款为用户提供快速小额贷款的信贷产品。用户随时随地都可申请贷款。最快三分钟即可完成审批和放款，解用户燃眉之急。根据用户个人情况，提供1000-5000元的借款额度，更贴心配置灵活还款服务，用户可个性化选择还款周期及还款方式。
  2. “马上钱包”主要向具有稳定、持续收入的客户提供个人信用循环额度贷款，可循环使用。根据客户的社保缴纳情况最高可为客户提供5万提现额度，是用户个人的信用钱包。随时随地，灵活提现，自由还款。
  3. “商品贷”是用户在特定消费场景使用的分期消费贷款产品。用户选定商品后，通过“马上金融”可申请商品分期贷款服务。最高可申请20万，分期期限可依据用户需求私人定制。你购物，我买单，消费更轻松~

产品定位

马上金融，幸福提前—马上消费金融致力于为中国国内居民提供更为便捷的个人消费金融服务。

功能分析

产品架构

先上主要功能结构图，为了方便清晰观看，去掉了一些次要分类：

核心功能

登录

手机号+验证码登录

马上金融支持两种登录方式：手机号+密码和第三方快捷登录（微信、支付宝）。登录流程很简单，就是输入手机号和密码后进行相关判断，如果手机号和密码都正确则登录app。

  1. 输入手机号，手机号是数据键盘，有位数及合法性校验；
  2. 输入密码，密码有8-16位以及数字字母组合的校验；
第三方快捷登录
  1. 点击第三方快捷登录icon，打开第三方app；
  2. 授权成功后，返回马上金融app；
  3. 开始账号绑定，输入手机号和密码；
改进建议

  1. 登录能够支持到验证码登录，目前大部分主流app都已经支持了验证码登录，这样的登录方式更便捷，也更符合人性。优化后的登录界面应该首选验证码登录，附近能够快捷切换到密码登录。

  2. 支持验证码登录后，这样注册以及第三方快捷登录的验证也可以更简单，没有注册的用户（第三方快捷登录的用户）使用手机号+验证码认就默认为注册（绑定）用户。

  3. 目前登录的时候没有检验手机号是否已经注册，所以录入未注册的手机号登录，提出是手机号及密码不匹配，提示不够清晰，可以在收入手机号后判断是否注册，如果注册可以引导用户注册。

  4. 更换手机号的功能建议在登录页面隐藏，放置在设置页面中。

  5. app 第一次下载后会申请获取权限，建议除非必要的权限，否则不要申请，因为弹窗特别多，很烦。刚刚注册就要获取相机权限，这个不合理，被拒绝的概率比较大，拒绝后实名认证的时候得让客户去手机系统设置里更改，操作成本大一些。应该在后面实名认证的时候在获取，这个时候获取权限通过率也会高一些。有些权限的申请和场景契合，用户更容易授权。

  6. 手机号+验证码登录一般流程（竞品招联金融）：

注册

目前马上金融app的仅支持手机号+密码的登录方式，注册要素基本都堆积在一个页面上，注册流程大概如下：

  1. 输入手机号，手机号是数据键盘，有位数及合法性校验。但是手机号没有判断是否已经注册过，已经注册过的用户还可以往下走，现实这样的情况比较少出现，但是建议可以直接判断是否注册，如果已注册过则告知用户并前往登录。
  2. 输入密码，密码有8-16位以及数字字母组合的校验。有密码强弱的设计，及时反馈给用户，很人性化。
  3. 获取验证码，点击后先获取图像验证码，在发送短信验证码。这个应该是处于安全的考虑。
  4. 图形验证码，图形验证码有刷新机制，比较注重安全。
  5. 某人勾选相关政策和协议。
  6. 输入完手机号、密码、图形验证码以及短信验证码，注册成功。如果手机号已经注册，此时提示我账号已注册，是否直接登录，用户体验不佳。
改进建议

  1. 建议页面拆分，拆分成录入手机号页面，输入验证码页面和设置密码等三个页面，这样给用户的认知负荷比较小，每个页面专注于一件事儿，转化率可能会高一些（可以进行埋点或者a/b测试）。

  2. 注册一开始时输入手机号就校验是否注册，如果注册了则直接引导去登录。这样总比输入完手机号、密码、验证码之后在告诉用户的要好，毕竟现在的流程设计，用户成本比较高。

  3. 取消勾选后，注册按钮还是高亮显示，这个时候点击又提示我要阅读并勾选，反馈的效果不佳。建议按钮一开始就是禁用置灰，输入框获取焦点后高亮。勾选的逻辑也是这样，取消候选后按钮禁用置灰，勾选够则高亮显示。

  4. 手机号+验证码注册一般流程（竞品招联金融）：

实名认证

实名认证是借款的前置任务，遵循业务成熟的四要素认证（姓名+身份证号+银行卡+手机号），大概流程如下：

  1. 验证身份信息，输入姓名及身份证号码；
  2. 验证银行卡信息，输入卡号、预留手机号及验证码；
改进建议
  1. 马上金融 app 会员中心 里有任务的设置，完成相关设置可以奖励积分，但是入口较深，建议将任务模块放到首页，这样用户一旦注册，就可以在首页看到任务，完成后还有奖励，可以提高认证的可能。
  2. 身份证号可以做相关插件，提供输入效率。
  3. 卡号基本都是数据，直接使用数字键盘即可；银行卡号如果能按照xxxx xxxx xxxx xxxx xxx 这样展示就更好；19位的时候卡号换行了，应该能够一行展示完成。
  4. 手机号码也是全键盘输入，应该跟注册登录统一使用数字键盘。
  5. 建议将实名认证拆分成步骤，通过进度条显示目前所在的步骤以及余下的步骤。这方面可以参考券商的开户流程—富途证券账户开通（股票的开户流包括了身份验证、银行卡验证、资料填写、风险披露、拍照验证、签字等。），富途证券账户流程大概拆解成以下六个步骤，每个步骤只完成一件事，并且有进度条显示，透明友好。
  • 上传身份证正反面，上传完成后反显识别出的信息，用户确认无误后进入到下一步。
  • 银行卡验证
  • 资料填写
  • 风险披露
  • 拍照验证
  • 签字

借款（暂时无法体验）

马上金融借款之前需完成个人信息认证，包括本人身份证和银行卡信息，选择进入“借款”页面，一共有3种借款产品：现金分期、循环额度和商品贷款（根据我的借款中的tab可以看出）。

由于体验时机问题，体验时马上金融借款因为资源整合，无法借款，文案提示页面的排榜比较差，可以进行优化。

商城

现今分期付款、提前消费已经慢慢成为一种趋势，很多电商平台也推出了分期付款的服务。马上金融把【商城】作为一个独立的版块，提升了部分用户的消费体验。从商城商品的分类来看，主要是数码电子产品、潮流服饰和女性美妆奢侈品等。跟常见的电商平台一样，【商城】版块也推出很多的促销活动，例如今日限时抢购、免息分期、优惠券等。

同时，商城也有充值中心，提供更多的生活充值服务。

进入到商城的二级页面后，底部tabbar还存在，可以快捷的前往其他板块。

还款（未体验）

用户及时还款是消费信贷类app需要首先考虑的问题，提高还款效率能有效提升用户体验。

其他功能

客服、消息

首页和个人信息页面都设有“消息“和“客服”入口按钮（在首页客服功能出现了两次），用户可以很方便找到，及时为用户解决问题的客服通道和及时的消息通知，能让用户的问题第一时间得到解决，有效提升用户体验。

个人信息

个人首页都主要包含了我的借款、我的还款、我的订单、个人信息认证（身份证、银行卡等）等主要信息，用户可随时查看和修改相应的信息。

改进建议
  1. 我的tabbar中的很多功能需要登录之后才可以使用，所以建议点击我的tabbar的时候判断用户是否登录，如果未登录则直接跳往登录页面，登录成功后跳往我的页面。（或者通过a/b测试来进行数据跟踪，目前的形式和未登录时点击我的直接登录，那个数据表现好一些）
  2. 设置按钮放置推荐放在右上角，与消息按钮并排，目前放置在底部。

总结

从公开数据看，2019消费金融公司排行榜中，马上金融位居第二，服务超过6500万用户，市场份额还在不断扩大。从融资能力方面看，马上金融已经拥有多元化的融资能力，除了6大股东存款有力支持，还与400家银行建立合作关系可以拆借资金，此外，马上金融的相关ABS产品即将发行，而相关金融债也在积极筹备，在消费金融领域，马上金融野心勃勃。

从商业模式和竞争态势上来看

从商业模式和竞争态势上来看，马上金融对盈利模式的逻辑还是很清晰的—核心还是利率差，能够持续获取多元且低廉的资金，通过互联网获客，在风控安全范围来发放贷款：

  1. 消费者提出分期或者先消费后付款的消费金融服务需求；
  2. 在征信评估和必要的风控模型下，服务商根据消费者所获信用发放消费额度；
  3. 服务商从资金供给方获得资金，提前为消费者付款给供给方；
  4. 供给方拿到资金后，发放商品/服务给消费者；
  5. 消费者分期付款/到期付款（包括利息）给服务商，此阶段服务商达到获利的商业目的。

同时要担心的也是最关键的一点，就是如何持续获取低廉的资金，如何持续获取优质的客户（目前的流量价格比较高昂）。

从整个体验的过程来看，目前马上金融app还没有引进内容系统，这方面可以进行建设，比如购买手机后需要哪些配件，或者根据用户画像来推送相关的内容。

最后

用户体验是一项系统性工作，需要PM、设计师、开发者、运营人员群策群力，以及不断地摸索、思考、改进，需要兼顾5个层级设计的一致性和关联性，绝非一朝一夕之功。

当然，由于自身认知水平、思维方式、产品知识积累的局限，很多地方还存在不足，万望指正。
